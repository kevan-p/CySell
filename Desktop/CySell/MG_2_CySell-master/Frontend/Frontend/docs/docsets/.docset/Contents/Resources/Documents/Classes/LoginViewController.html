  • LoginButtonClicked(_:)

    When the Login button is clicked, Facebook goes through it’s authorization. If the case is failed, then it prints an error, if the user cancels their login, then a message prints User cancelled login. If the login is a sucess, it grabs the User data from facebook and posts it in the database. Then it transitions into the main page of the app.
